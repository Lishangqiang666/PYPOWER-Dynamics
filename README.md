PYPOWER-Dynamics
================

Dynamic time-domain simulation module for PYPOWER.
Note: this is a very preliminary commit. Use at your own risk!

Dependencies
============

PYPOWER-Dynamics requires:

* Python_ 3+
* PYPOWER_
* SciPy_ stack 0.9 or later.
